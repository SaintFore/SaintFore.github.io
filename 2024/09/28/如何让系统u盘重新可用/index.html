如何让系统u盘重新可用

本文最后更新于 2024年9月28日 下午

引言

我们将 u 盘制作为系统 U 盘后，U 盘就没法在电脑中正常识别出了。当装完系统，不再需要 u 盘充当系统 U 盘想要正常使用该 U 盘，这时候就需要有些操作，让这个 U 盘正常化。

上图就是充当系统盘的 u 盘，无法正常使用。

开始操作

还是 win + R，输入 cmd。

输入 diskpart 指令，回车，点击确定。接着就会出现下面的界面。

接着输入，list disk。就是显示电脑安装的所有磁盘。

看大小其实就知道哪个是 U 盘，显然最小的就是 U 盘。

使用 select disk 磁盘号 指令来选择磁盘，一定要选对，记住信息无价，弄错了就 GG 了。

选择对应的磁盘。

输入 clean 指令。

这时候其实电脑还是看不到 U 盘的，因为还没有格式化。

输入 create partition primary 来创建分区，然后 list partition 来查看分区，接着 select partition 分区号 来选择分区。

最后输入 format quick 进行格式化。

这时候我们打开此电脑就能看到能够正常使用的 U 盘了。

遇到的错误

输入 clean 时候也可能遇到上面的问题，这是因为你提早关闭了 cmd 窗口，也就是下面这个，再次重复上述步骤，但是别管 cmd 窗口，就能解决问题。

工具 > U盘
