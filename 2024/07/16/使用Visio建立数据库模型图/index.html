使用Visio建立数据库模型图

本文最后更新于 2024年7月17日 晚上

为了方便建立数据库的模型图，我们可以使用 Visio 的一个模板，使用它的方向工程，能够快速建立数据模型的图片，非常方便。

MySQL Connector

有些电脑在安装 MySQL 的时候不安装 MySQL Connector，这导致之后加入 ODBC 数据源的时候只有一些微软自带的数据库，例如 SQL Server，但是没有 MySQL 的选项。

我们可以通过 Installer 来进行安装 MySQL Connector。

先打开 MySQL Installer，点击右边的 Add。

选择 Connector/ODBC，选择自己的版本，一般六十四位，这个 Connector 的版本跟 MySQL 的版本是不同的，只用管是 8.0 还是 5.7，这个得一样，小版本不用管，可以不同。

通过 Visio 来建立数据模型图片

选择数据库模型图，注意用管理员模式打开。

进入后，点击上面的数据库，选择反向工程，选择数据源，然后就可以了。

注意选择 ODBC。
