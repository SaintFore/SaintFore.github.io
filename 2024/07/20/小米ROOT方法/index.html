小米ROOT方法

本文最后更新于 2024年7月20日 中午

因为涉及到了需要用到 root 的手机，所以又拿出了我的老手机小米 5 来完成我的需求。

因为小米，为发烧而生，所以它的 ROOT 相当简单，虽说我花费了不少的时间。

解 BL 锁

手机设置

首先就是解开 BL 锁。

在设置 -> 我的设备 -> 全部信息与参数里面，先进入。

连续点击 MIUI 版本，进入开发者模式。

找到开发者选项，这个一般在更多设置里面。

打开开发者选项，还有 OEM 解锁。

同时还要打开 USB 调试。

除此外要点击一下设备解锁状态，进入页面，现在显示的是当前设备已经锁定，如果你还没有解开 BL 锁的话，点击绑定账号与设备。

最后，关机。

等待关机后，声音减号键加上关机键一起按下进入 fastboot 模式。

到这里，手机的设置就结束了。

电脑下载

申请解锁小米手机 (miui.com) 进入该网站，下载 miflash_unlock 软件，这是一个 zip 压缩包，解压后点击 exe 文件，进入操作页面

之后点击小齿轮，先安装驱动。

点击这个驱动检测，如果没安装驱动，它会自动帮你安装。

安装成功也有可能不显示成功，不管就行。

返回操作界面，使用 usb 连接手机与电脑，点击解锁。

可能不成功，例如说没账号或者时间不够之类的，没账号可能是因为需要给手机安装 sim 卡，时间不够就是要求你绑定账号时间得达到七天。

如果都没有遇到，等到进度条走到 100，你就解锁成功了。

解锁成功

现在设备解锁页面的图片就是上图所示这样。

刷机

下载刷机包

也就是下载小米 rom，进入下面网站进行下载，XiaomiROM.com - 小米 ROM | MIUI、澎湃OS（HyperOS）线刷包, 卡刷包的最新及历史版本下载

选择你手机的版本，如果不恰当有可能变砖，后果就只能自负了。

下载的 rom 包也是一个 zip 压缩包，解压，找到 boot. img，这就是所需要的文件。

使用 magisk 进行修补

这一步只是为了完整的安装 magisk，也就是为了让引导分区里面有 ramdisk，可以看看安装手册，因为有些好玩的内容需要用到 magisk 的模块，所以我就一并做了，不然之后还得再重复一次。

拿到 boot. img 传到手机里面。

在手机中安装 magisk，先下载 magisk，点击 releases，在 assets 中选择下载 apk 就行。

把这个文件传到手机里面，进行安装，这里是没有开启 ramdisk 的。

所以需要进行修补。

进入 magisk 应用，选择安装。

直接下一步。

选择第一个选项，然后选择刚刚传到手机里面的 boot. img 文件，开始修补，修补完后会在 download 文件夹中生成一个 magisk_patched 文件开头的 img 文件，这就是我们要刷进手机里面的文件。把它传到电脑里面。

开始刷机

下载秋之盒，也是一个压缩包，解压即用。

进入该页面，然后连接手机，注意手机仍然要进入 fastboot 模式，也就是先关机，同时按下减号以及关机键。连接后，就 fastboot 刷入，选择刚刚传进电脑里面那个 magisk_patched 开头的 img 文件，等待进度条跑满。

然后就成功了。这里也没有清空手机文件，只是 root 了手机。

现在就可以开启 root 模式了。

告成

现在就成为超级用户了，可以使用解除许多限制，玩点好玩的了。
